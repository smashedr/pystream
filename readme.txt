Can be run manually and enter in username or URL.

Can be used programmatically by passing in the username or URL as an argument.

Usage:
pystream.exe [username/URL]

Example:
pystream.exe bobross

This requires Microsoft Visual C++ Redistributable or you get an error about "MSVCP140.dll".
https://support.microsoft.com/en-us/help/2977003/the-latest-supported-visual-c-downloads
